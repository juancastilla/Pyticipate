Installation and Run instructions for SimCopiapo on Windows [64-bit]

INSTALLATION:

 1) Install miniconda (found in the Installers\Miniconda directory)

 2) Start Anaconda Prompt (can be found in Start Menu at Anaconda3/Anaconda Prompt)

 3) Navigate to the SimCopiapo directory

 4) Run INSTALL.BAT (requires internet access to download Python packages)

RUN:

 1) Start Anaconda Prompt (can be found in Start Menu at Anaconda3/Anaconda Prompt)

 2) Navigate to the SimCopiapo directory

 3) Run RUN.BAT (product should be up and running in less than ten seconds)
